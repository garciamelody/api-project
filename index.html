Fire Emblem Awakening Characters

The unofficial API for all characters (including NPCs) in the game Fire Emblem Awakening.

To access objects of individual characters use: https://fire-emblem-api.herokuapp.com/api/character name

example:https://fire-emblem-api.herokuapp.com/api/chrom

For the JS formatted fetch parameter use: `https://fire-emblem-api.herokuapp.com/api/${character name}`

The Properties

There are 3 properties that come with each character and each return a string

  • startingClass

    • The class at which the chatacter starts in the game

  • birthPlace*

    • The country or town that the character was born in

  • weapon

    • The main weapon that a character uses / starts out with

* most npcs won't have a birthPlace due to that information being unknown
